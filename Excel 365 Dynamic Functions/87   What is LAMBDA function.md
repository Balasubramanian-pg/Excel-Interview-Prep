### 87. **What is LAMBDA function?**

Creates custom reusable functions:
Syntax: =LAMBDA(parameter1, parameter2, ..., calculation)

Example:
=LAMBDA(x, y, x^2 + y^2)

**Must be saved as named formula, then used:**
If named "Pythagorean": =Pythagorean(3, 4) returns 25
