### 86. **What is LET function?**

Assigns names to calculation results for reuse:
Syntax: =LET(name1, value1, name2, value2, ..., calculation)

Example:
=LET(x, A1*2, y, B1*3, x+y)

**Benefits:**

- Cleaner formulas
- Better performance (calculates once)
- Easier debugging
